# macterial
A mac inspired GTK+ theme based on Material design only for GNOME

Released 19 July 2018


### How to use?
1. Create a folder named '.themes' in your home directory
```
mkdir /home/user_name/.themes
```

2. Download the '.zip' file and extract it to the '.themes' directory

3. Install gnome-tweak-tool
```
sudo apt install gnome-tweak-tool
```

4. Run gnome-tweak-tools and select the theme as 'Macterial'
